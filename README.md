# Projet pfSense - Firewall
## Objectif
Mettre en place un firewall pour sécuriser un réseau.
## Environnement
- Virtualisation avec VMware
- pfSense
## Étapes
- Création d’une machine virtuelle sur VMware
- Installation de pfSense
- Configuration des interfaces WAN / LAN
- Mise en place de règles de filtrage
- Test de connectivité (ping)
## Résultat
Le firewall fonctionne correctement et permet de contrôler le trafic réseau.
## Ce que j’ai appris
- Fonctionnement d’un firewall
- Configuration réseau de base
- Utilisation de la virtualisation (VMware)
